# 🚀 Accredian Enterprise Landing Page

A modern, responsive, and pixel-perfect **Enterprise Landing Page** built as a frontend assignment inspired by the Accredian Enterprise website. The project focuses on clean UI, reusable React components, responsive layouts, and a smooth user experience using **Next.js**, **TypeScript**, and **Tailwind CSS**. The repository uses a component-based architecture with separate sections for the hero, statistics, clients, domain expertise, FAQs, testimonials, and more. :contentReference[oaicite:0]{index=0}

# 💻 Getting Started

## Clone the Repository

```bash
git clone https://github.com/shivamvermajss/Accredian-Enterprise-Page.git
```

## Navigate to the Project

```bash
cd Accredian-Enterprise-Page
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```

---

# 📦 Build for Production

```bash
npm run build
```

Start Production Server

```bash
npm start
```
